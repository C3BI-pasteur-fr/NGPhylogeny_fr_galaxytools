NGPhylogeny.fr OneClick Workflows

We have designed several oneclick workflows to address different kinds of analysis. They all include the same steps:

  1. Sequence alignment
  2. Alignment curation/filtering
  3. Tree inference
  4. Tree visualisation

These workflows integrate the gold-standard software for each step:

1 - Multiple Sequence alignment (MSA) is performed using MAFFT [1] It's an excellent compromise between speed and accuracy. Pais, P. et al., 2014 [2]

2 - Alignment curation is performed using Noisy [3] which is able to remove homoplastic or otherwise corrupted sites from putative phylogenetically informative sites. By default options have been set following recommendations of Tan, G. et al., 2015 [4].

3 - Tree inference is performed using PhyML [5] or FastME [6]. PhyML should be preferred to get results with a high level of accuracy while FastME should be preferred to get results in a short amount of time (in particular FastME can be preferred to deal with very big data sets although the results obtained will have lower accuracy).

4 - Tree visualization is obtained using the Newick utilities [7] both to draw the trees and provide pictures in SVG format.

Each workflow run (i) takes as input a fasta file containing the sequences (DNA or proteins) and (ii) produces several output files :

  • MSA in PHYLIP format
  • Cleaned MSA
  • Tree in Newick format
  • Tree picture in SVG format.

From the following sections, you can retrieve the workflows and install them in your galaxy.pasteur.fr account and analyse your data.
