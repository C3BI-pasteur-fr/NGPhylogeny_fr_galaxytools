NGPhylogeny.fr OneClick Workflows

We designed several oneclick workflows to address different kind of analyses. They all include the same steps:

  1. Sequence Alignment
  2. Alignment curation/filtering
  3. Tree inference
  4. Tree visualisation

Theses workflows integrate the gold standard software for each step:

1 - MAFFT [1] is used to align sequences and get a Multiple Sequences Alignment (MSA). The choosen options provide an excellent compromise between speed and accuracy(see: Pais,P.al, 2014)[2].

2 - Noisy [3] is used to remove poorly aligned regions within the MSA. The options are set according to (Tan et al., 2015)[4].

3 - PhyML[5]and FastME[6] are available to infer a tree. PhyML is very accurate but is a very CPU consuming while FastME runs very fast at the cost of accuracy. FastME is relevant only for the biggest datasets.

4 - The Newick utilities [7] are used to draw the trees and provide nice pictures in SVG format.

Each workflow takes as input a fasta file containing the sequences (DNA or proteins). Each run will produce several files:

  • MSA in PHYLIP format
  • Cleaned MSA
  • Tree in Newick format
  • Tree picture in SVG format.

From the following sections, you can retrieve the workflows and install them in your galaxy.pasteur.fr account and analyse your data.
